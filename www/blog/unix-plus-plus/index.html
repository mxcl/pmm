UNIX++ Pack

UNIX++ Pack.

Modern UNIX command line replacements and operators for search, file inspection, process monitoring, data wrangling, and HTTP/DNS work.

Install the pack Package list

The classic UNIX toolchain still works, but agent work benefits from tools that are faster, more readable, and easier to compose. Operators need output that humans can scan and agents can parse.

UNIX++ Pack turns the old pack idea into a public shortlist: 23 Homebrew packages for finding, previewing, navigating, watching, measuring, transforming, and querying from the command line.

Why this pack exists

shell layer
01 / inspect

Make local state easier to read.

bat, eza, dust, duf, procs, bottom, and tokei make files, directories, disks, processes, and code volume more visible.

02 / navigate

Move through projects faster.

fd, ripgrep, fzf, zoxide, choose, and sd reduce shell friction when operators and agents are exploring a tree.

03 / operate

Turn shell output into decisions.

jq, yq, miller, xh, doggo, entr, watchexec, git-delta, and hyperfine help parse, query, watch, diff, and measure work.

Pack install

Install these 23 UNIX++ packages from the app.

The pack link sends every package as one install request. pkg⋅mgr² still installs them one at a time through the existing package managers.
