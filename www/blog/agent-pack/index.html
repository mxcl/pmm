Agent Pack

Agent Pack.

Agent CLIs and coding assistants for terminal-native planning, editing, review, model routing, and usage inspection.

Browse the tools Package list

Agent work no longer happens inside one app. Operators jump between coding assistants, model routers, usage inspectors, and terminal-native CLIs that can plan, edit, review, and explain a repository from the shell.

Agent Pack lists those tools for Package Manager Manager users. Use it when a workstation needs the practical agent surface without a separate pack installer.

Why this pack exists

agent layer
01 / assistants

Run the coding agents developers use.

Codex, Claude Code, Goose, Aider, OpenCode, Gemini CLI, and Qwen Code cover the common terminal-native agent entry points.

02 / routing

Keep model access scriptable.

llm and mods make prompts, providers, and shell pipelines easier to compose without leaving the terminal.

03 / accounting

Know what the agents cost.

ccusage gives operators a quick view of Claude Code token and cost usage before a run grows invisible.
