Agentic Toolkit

The Agentic Toolkit.

All the tools agents need: media processing, image manipulation, runtimes, search, shell, build, OCR, metadata, and document conversion.

Install the pack Package list

Agents inspect repositories, render media, parse structured files, build native projects, read screenshots, and turn documents into something a model can reason about. That means a useful local agent environment needs more than one assistant CLI.

The Agentic Toolkit is a Package Manager Manager shortlist: 26 Homebrew packages that cover the practical surface area agents reach for when they work on a real machine. Use the package list below to inspect and install the tools you want.

Why this pack exists

operator layer
01 / assets

Agents need to see and change media.

Image, video, PDF, OCR, metadata, and document conversion tools make non-code work visible.

02 / runtime

Agents need the runtimes projects use.

Node, Python, Deno, Go, shell, package managers, and build metadata cover the common execution paths.

03 / review

Agents need fast inspection tools.

Search, file finding, code previews, diffs, formatting, linting, and benchmarks keep the operator in control.
