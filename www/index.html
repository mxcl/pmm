From install to source

See the package, not just the package name.

Open a package to see its installed version, install location, binary path, homepage, repository, and docs. When pkg⋅mgr² knows the native command, update or uninstall it from the same view.

  • Installed and latest versions
  • Project links and local paths
  • Native update and uninstall actions

One app, more Macs

Update the Macs you already SSH into.

Add a Mac from the Package Manager Manager menu. Its Installed and Outdated lists appear in the sidebar, ready for updates and uninstalls.

  • Local network and Tailscale hosts
  • Your existing SSH config, keys, agent, and known_hosts
  • Individual updates, Update All, and uninstalls
